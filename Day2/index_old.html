Part 2: Neural Networks 🧠

How computers learn like brains!

What is a Neural Network?

The Basics:

A computer program inspired by how your brain works!

  • Made of artificial "neurons" (like brain cells)
  • Neurons connect to each other
  • They pass information like a relay race

🧠 Real Brain vs Computer Brain

Your Brain

100 billion neurons! 🤯

  • Cells that send signals
  • Connect through synapses
  • Learn by strengthening connections
Neural Network

Mathematical neurons 🔢

  • Numbers and equations
  • Connect through "weights"
  • Learn by adjusting numbers

🏗️ Structure of a Neural Network

Three Types of Layers:
Input Layer

Where data enters
👁️ 👂 📝

Like your senses
→
Hidden Layers

Where thinking happens
🤔 💭 🧮

The "brain" part
→
Output Layer

The answer
✅ ❌ 🎯

The decision

Visualizing Neural Networks

🔄 Neural Network Workflow - Step by Step

Step 1:
Input Data

Feed in numbers (pixels, sounds, text converted to numbers)

➡️
Step 2:
Forward Pass

Data flows through network, each neuron does math

➡️
Step 3:
Output Prediction

Network makes a guess (70% cat, 30% dog)

➡️
Step 4:
Check Answer

Compare to correct answer, calculate error

➡️
Step 5:
Learn (Backpropagation)

Adjust weights to reduce error next time

How Neural Networks Learn

The Training Process:
  1. Dataset: Thousands of examples with correct answers (labels)
  2. Epochs: Going through all data multiple times
  3. Loss: How wrong the network is
  4. Optimization: Making the network less wrong
Like learning math: Practice problems → Check answers → Learn from mistakes → Get better!

Example: Teaching a Network to Recognize Cats

  • Collect Data: 10,000 cat photos, 10,000 non-cat photos
  • Prepare: Convert each photo to numbers (pixels)
  • Train:
    • Show photo → Network guesses → Tell if right/wrong
    • Repeat thousands of times
    • Network adjusts to get better
  • Test: Show new photos it's never seen
  • Success: 95% accuracy! 🎉

Why Do Neural Networks Work So Well?

Three Key Reasons:
  1. Universal Approximators: Can learn any pattern (with enough neurons)
  2. Hierarchical Learning: Simple features → Complex understanding
    • Layer 1: Edges and lines
    • Layer 2: Shapes
    • Layer 3: Objects
  3. Lots of Data: Internet provides millions of examples

Activity: Be a Neural Network!

Human Neural Network Game (10 minutes):

  1. Form groups of 5–6 people
  2. Each person is a "neuron"
  3. Pass a number, multiply by your "weight" (a number like 2, 0.5, –1, etc.)
  4. Last person announces if result > 10
  5. Adjust weights to get correct answers!

This shows how neural networks learn through adjustment!

Part 3: Computer Vision 👁️

Teaching computers to see and understand!

What is Computer Vision?

Definition:

The field of AI that enables computers to understand and interpret visual information from the world

Human Vision: Eyes capture light → Brain understands "that's a dog"
Computer Vision: Camera captures pixels → AI understands "dog detected"

How Computers "See" - It's All Numbers!

Images are Made of Pixels

Each pixel has a number

Image source: Stanford AI - Serena Yeung's Computer Vision Course

🎨 Understanding Color in Images

RGB Color Model

Every color = Mix of Red, Green, and Blue

Each from 0 (none) to 255 (full)

Red = (255,0,0)

Green = (0,255,0)

Blue = (0,0,255)

Yellow = (255,255,0)

🔄 Computer Vision Workflow

From Image to Understanding:

1. Image Capture

Camera takes photo → Pixels stored as numbers

2. Preprocessing

Resize, normalize, enhance quality

3. Feature Extraction

Find edges, shapes, textures, colors

4. Pattern Recognition

Neural network identifies objects

5. Decision/Output

"Cat detected with 95% confidence"

🧠 CNNs: The Champions of Computer Vision

Convolutional Neural Networks

Special neural networks designed for images!

How CNNs Work:
  1. Convolution: Scan image with filters to find features
  2. Pooling: Shrink image while keeping important parts
  3. Repeat: Multiple layers find complex patterns
  4. Classify: Final layers decide what's in the image

Making AI Smarter: Data Augmentation

Teaching AI to handle variety:

Take one image and create many versions!

Original photo of cat 🐱 becomes:

  • Rotated cat 🔄
  • Flipped cat 🔁
  • Brighter/darker cat 🔆
  • Zoomed cat 🔍
  • Slightly blurred cat 🌫️

Now AI learns cats can look many ways!

🎓 Transfer Learning: AI Sharing Knowledge

Don't start from scratch!

Use a model that already knows basic vision

Like learning a new sport:
If you can play tennis 🎾, learning badminton 🏸 is easier!
You already know how to swing and move!
  1. Take model trained on millions of images
  2. Keep the "seeing" parts
  3. Train only the "deciding" parts for your task
  4. Much faster and needs less data!

🔧 Complete Computer Vision Pipeline

Example: Building a Dog Breed Classifier

1. Collect Data

1000 photos each of 50 dog breeds

2. Preprocess

Resize to 224x224, normalize colors

3. Augment

Create variations: flip, rotate, zoom

4. Choose Model

Use pre-trained ResNet, modify last layer

5. Train

50 epochs, monitor accuracy

6. Evaluate

Test on new photos, 94% accuracy!
